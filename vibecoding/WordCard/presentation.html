Overview

프로젝트 목표

📖

단어 학습 도구

영어 단어, 발음기호, 한국어 뜻, 예문을 카드 형태로 시각화하여 효율적인 단어 학습을 지원합니다.

🔍

LIKE 검색

단어·뜻·예문·발음기호 4개 필드를 동시에 LIKE 검색하여 빠르게 원하는 단어를 찾을 수 있습니다.

🗂️

정렬 기능

최신순(등록일 DESC) 또는 알파벳순(단어 ASC)으로 목록을 정렬하여 원하는 순서로 학습합니다.

➕

단어 추가

모달 폼을 통해 새로운 단어를 DB에 즉시 저장합니다. POST → Redirect → GET 패턴을 따릅니다.

Feature 1

LIKE 검색

  • 영어 단어, 한국어 뜻, 예문, 발음기호 4개 필드 동시 검색
  • 대소문자 구분 없는 부분 일치 검색 (LOWER LIKE)
  • 검색 버튼 클릭 또는 Enter 키로 실행
  • 검색 결과 건수와 전체 건수 함께 표시
  • 매칭 텍스트 노란 하이라이트 강조

⚠️ Hibernate 6 이슈 해결

CONCAT('%',:q,'%') → ESCAPE '' 버그
Java에서 "%" + q + "%" 전달로 해결

📝 WordRepository.java

@Query(""" SELECT w FROM Word w WHERE LOWER(w.word) LIKE :q OR LOWER(w.ko) LIKE :q OR LOWER(w.example) LIKE :q OR LOWER(w.phonetic) LIKE :q """) List<Word> searchByQuery( @Param("q") String q, Sort sort);

📝 WordService.java

// Java에서 % 직접 전달 String like = "%" + q.toLowerCase() + "%"; wordRepository.searchByQuery(like, sort);
Feature 2

정렬 기능

🕐

최신순 (newest)

가장 최근에 등록한 단어가 먼저 표시됩니다.

Sort.by(Direction.DESC, "createdAt")
🔤

알파벳순 (alpha)

단어를 A–Z 알파벳 오름차순으로 정렬합니다.

Sort.by(Direction.ASC, "word")

Feature 3

단어 추가

  • 우하단 ＋ FAB 버튼으로 모달 열기
  • 영어 단어, 품사, 발음기호, 뜻, 예문 입력
  • @NotBlank 유효성 검사 (단어·뜻 필수)
  • POST /words → DB 저장 → redirect:/
  • PRG 패턴으로 새로고침 중복 방지

PRG 패턴 흐름

1️⃣ POST /words (폼 제출)
2️⃣ DB 저장 (wordRepository.save)
3️⃣ redirect:/ (302 응답)
4️⃣ GET / → Thymeleaf 렌더링

📝 WordController.java

@GetMapping("/") public String index( @RequestParam String q, @RequestParam String sort, Model model) { model.addAttribute("words", wordService.searchAndSort(q, sort)); return "index"; } @PostMapping("/words") public String addWord( @Valid @ModelAttribute WordForm form) { wordService.save(form); return "redirect:/"; // PRG }
View Layer

Thymeleaf 템플릿

  • th:each — 단어 카드 반복 렌더링
  • th:utext — 하이라이트 HTML 출력
  • th:action="@{/words}" — POST 폼
  • th:classappend — 정렬 버튼 active 상태
  • th:if / #strings — 조건부 렌더링
  • 모든 단어 카드는 서버(DB)에서 렌더링

📝 index.html (카드 반복)

<!-- 단어 카드 --> <div th:each="word : ${words}" class="word-card"> <div class="word-en" th:utext="${word.wordHtml()}"> </div> <span class="word-pos" th:text="${word.posLabel()}"> </span> <div th:utext="${word.koHtml()}"></div> <div th:utext="${word.exampleHtml()}"> </div> </div>
