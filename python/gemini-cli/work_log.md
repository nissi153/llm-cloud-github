## 2. ex02.py - 사칙연산 프로그램
간단한 사칙연산(덧셈, 뺄셈, 곱셈, 나눗셈)을 수행하는 파이썬 스크립트 작성 및 실행 완료.
```python
a = 10
b = 5

print(f"{a} + {b} = {a + b}")
print(f"{a} - {b} = {a - b}")
print(f"{a} * {b} = {a * b}")
print(f"{a} / {b} = {a / b}")
```
